
# Baria

Sebuah single page application dengan tujuan untuk belajar huruf jepang

## Installation

```bash
  git clone https://github.com/Rxnel-ysr/Baria
  cd Baria
  npm run dev
```

karna project bisa berjalan secara local dan sama sekali tidak memerlukan depedency tambahan, project bisa langsung dijalankan via live server VSCode atau Hosting file static.
